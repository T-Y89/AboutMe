## SmartHR

期間: 2025/08/01~現在
所属: 品質保証本部
職種: QAE（SET, TEも含む）

### 主にやっていること

- インプロセスQA
  - プロダクト開発チームの一員として、品質とデリバリーの両立を支える仕組みづくり
  - リスク分析、テスト戦略、テスト計画、テスト設計レビュー
  - 自動テスト（ユニット・統合・E2E）の設計・実装・レビュー
  - CIのコスト最適化
  - プロダクトエンジニア / PM / CS / Designer と協働したQA活動のリード
  - 不具合分析を起点にした改善アクションの設計
  - AIを活用した品質保証の導入・運用

- 組織を横断する施策のリード
  - AIを活用しテストスイート・テストケースを資産にする仕組みの設計・開発チームへの普及
  - 品質データを継続的に収集する仕組みの設計
  - 集めたデータを元に分析し定量的にリスクを判断・検知できる仕組みの設計
  - これらを元にPM・PO・エンジニアリングマネージャーと意思決定をする

- 社内AI推進組織のメンバーとしてのAI活用推進
  - 全社でAIの利用率を上げるための施策の計画・施行
  - コストの暴走の防止する仕組みの設計・施行
  - セキュリティと自由度のバランスを取る設定の計画・配布

- QAEの採用活動
  - 部署の状況を踏まえた採用計画の立案 
  - スカウト・書類選考・カジュアル面談・選考　を実施
